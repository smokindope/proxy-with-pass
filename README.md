# Proxy Dashboard Worker

One-click Cloudflare Worker + D1 app.

## Deploy

Click the button below:

[![Deploy to Cloudflare](https://deploy.workers.cloudflare.com/button)](https://deploy.workers.cloudflare.com/?url=https://github.com/smokindope/proxy-with-pass)

## Setup after deploy

1. Cloudflare will ask to create a D1 database
2. Run the provided `schema.sql` in the D1 UI editor
3. Done
